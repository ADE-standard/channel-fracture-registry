# CF-0097

## Title
Auto Function Invocation lacks runtime RBAC → unauthorized execution via indirect prompt injection

## Category
Credential scope bypass

## Affected System
Semantic Kernel

## Layer
L4

## Failure Mode
Missing Runtime Authorization Gate

## Trigger
@kernel_function decorated method invoked via Auto Function Invocation

## Observed Symptom
Tool executes without authorization check; indirect prompt injection triggers unauthorized operations

## Root Cause
No security attributes at @kernel_function metadata layer; no runtime mid-execution approval mechanism

## Detection
CADVP CC-6 credential scope boundary check on function invocation

## Impact
False belief of execution sandboxing; functions execute with full ambient permissions

## Mitigation
Add RBAC metadata + runtime approval hook to kernel_function decorator
