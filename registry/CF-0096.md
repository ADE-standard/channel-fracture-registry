# CF-0096

## Title
Google AI connector leaks thinking/thought text parts into ChatMessageContent → reasoning exposed

## Category
Env contamination

## Affected System
Semantic Kernel

## Layer
L2

## Failure Mode
Thinking Text Leakage to Application Layer

## Trigger
Gemini with thinking_config include_thoughts=true via Google AI connector

## Observed Symptom
Model's internal reasoning appears as response content; application sees thought text instead of actual answer

## Root Cause
Connector passes thinking text parts unfiltered into ChatMessageContent instead of isolating them

## Detection
CADVP CC-1 guard execution counter on content filtering boundary

## Impact
False belief of content isolation; internal reasoning exposed to downstream consumers

## Mitigation
Separate thinking/thought blocks from response content in connector pipeline

arxiv:2606.04896
