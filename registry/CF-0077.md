# CF-0077

## Title
DNS Rebinding bypass in SSRF protection; MCP tools bypass SSRF validation entirely

## Category
Credential scope bypass

## Affected System
CrewAI

## Layer
L4

## Failure Mode
SSRF Protection Bypass via DNS

## Trigger
MCP tool call with DNS-rebinded hostname

## Observed Symptom
DNS rebinding bypasses SSRF protection; internal endpoints accessible without alert

## Root Cause
SSRF validation checks against initial DNS resolution; rebinding after check bypasses all protections

## Detection
CADVP CC-6 credential scope boundary check with DNS rebinding simulation

## Impact
False belief of network isolation; internal systems exposed via DNS TOCTOU

## Mitigation
Validate against resolved IP at connection time, not resolution time

arxiv:2606.04896
