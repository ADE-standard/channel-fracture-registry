# CF-0058

## Title
Checkpoint serialization drops deque maxlen → bounded becomes unbounded after round-trip

## Category
Tool surface illusion

## Affected System
LangGraph

## Layer
L3

## Failure Mode
Constraint Erasure via Serialization

## Trigger
State with bounded deque checkpointed and restored

## Observed Symptom
Deque grows unbounded; memory leak after state restore

## Root Cause
Serializer preserves items but not maxlen configuration parameter

## Detection
CADVP CC-4 content hash + configuration fidelity

## Impact
Silent unbounded memory growth after state restore

## Mitigation
Preserve maxlen/collection constraints during serialization
