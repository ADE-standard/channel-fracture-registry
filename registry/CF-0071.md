# CF-0071

## Title
Structured output with raw JSON-schema dict never validated → ToolStrategy handle_errors unreachable

## Category
Hook not firing

## Affected System
LangChain

## Layer
L3

## Failure Mode
Validation Path Dead Code

## Trigger
Raw JSON-schema dict used for structured output

## Observed Symptom
Output never validated; handle_errors retry path unreachable; malformed output passes silently

## Root Cause
Raw JSON-schema dict bypasses validation entirely; ToolStrategy handle_errors never triggered

## Detection
CADVP CC-3 hook execution counter on validation path

## Impact
False belief of output validation; malformed structured data silently accepted

## Mitigation
Route raw JSON-schema through validation path

arxiv:2606.04896
