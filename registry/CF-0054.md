# CF-0054

## Title
checkpoints never flushed mid-session; data loss on non-graceful exit

## Category
Silent message loss

## Affected System
LangGraph

## Layer
L3

## Failure Mode
Checkpoint Flush Omission

## Trigger
Long-running langgraph dev server with SIGKILL/non-graceful exit

## Observed Symptom
Thread list survives but all checkpoints gone; GET /threads/{id}/state returns 500

## Root Cause
Checkpoints only written on close() clean shutdown; mid-session flushes never occur

## Detection
CADVP CC-2 config vs runtime diff

## Impact
False belief of durable state; all progress lost without awareness

## Mitigation
Implement periodic mid-session checkpoint flush

arxiv:2606.04896
