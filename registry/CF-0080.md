# CF-0080

## Title
adk api_server --a2a silently mounts no A2A routes → UnboundLocalError swallowed by try/except

## Category
Config override

## Affected System
Google ADK

## Layer
L4

## Failure Mode
Silent Configuration Failure via Exception Swallow

## Trigger
Running adk api_server --a2a with agent.json containing agent card

## Observed Symptom
Server starts without any A2A routes; CLI flag accepted but silently does nothing; only one ERROR log

## Root Cause
UnboundLocalError on 'json' in get_fast_api_app swallowed by try/except; A2A route setup fails silently

## Detection
CADVP CC-2 config vs runtime route verification

## Impact
False belief of A2A protocol availability; server runs without intended functionality

## Mitigation
Fix variable shadowing; fail loudly when route setup fails

arxiv:2606.04896
