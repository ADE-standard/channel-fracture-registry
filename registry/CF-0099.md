# CF-0099

## Title
chroma similarity_search returns raw distances instead of normalized relevance scores

## Category
Tool surface illusion

## Affected System
LangChain

## Layer
L2

## Failure Mode
Score Semantics Misrepresentation

## Trigger
similarity_search_by_vector_with_relevance_scores on Chroma vector store

## Observed Symptom
Relevance scores are raw distances; higher=worse instead of higher=better; ranking inverted

## Root Cause
Method returns unnormalized distance values from Chroma without converting to similarity scores

## Detection
CADVP CC-7 tool availability probe with known-item search validation

## Impact
False belief of correct ranking; search results in reverse order of relevance

## Mitigation
Normalize distance to similarity (1-distance) or document that scores are distances
