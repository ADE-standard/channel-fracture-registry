# CF-0067

## Title
ProviderCapabilities static fallbacks skipped when LiteLLM introspection fails

## Category
Config override

## Affected System
CrewAI

## Layer
L2

## Failure Mode
Capability Detection Fallback Skipped

## Trigger
LiteLLM provider introspection fails

## Observed Symptom
Static fallback never runs; agent operates with wrong capability assumptions

## Root Cause
Early-return guard before get_capabilities() skips fallback on introspection failure

## Detection
CADVP CC-2 config vs runtime capability diff

## Impact
False belief of accurate provider capabilities; agent behavior diverges

## Mitigation
Run static fallback before early-return guard

arxiv:2606.04896
