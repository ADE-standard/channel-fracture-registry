# CF-0098

## Title
ChatFireworks._combine_llm_outputs raises TypeError on nested token_usage dicts in batched generate()

## Category
Tool surface illusion

## Affected System
LangChain

## Layer
L2

## Failure Mode
Nested Metadata Parsing Crash

## Trigger
Batched generate() with Fireworks models returning nested prompt_tokens_details

## Observed Symptom
TypeError on token usage dict merge; batch generation silently fails

## Root Cause
_combine_llm_outputs expects flat token_usage dict but Fireworks returns nested structure

## Detection
CADVP CC-7 tool availability probe with batch generation

## Impact
False belief of batch compatibility; nested metadata silently crashes processing

## Mitigation
Recursively merge nested token_usage dicts in _combine_llm_outputs

arxiv:2606.04896
