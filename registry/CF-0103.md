# CF-0103: Long tool calls (~180s+) silently re-executed from checkpoint on LangGraph Cloud

| 字段 | 值 |
|------|-----|
| **CF-ID** | CF-0103 |
| **类别** | Silent re-execution |
| **影响系统** | LangGraph |
| **ADE层级** | L3 |
| **故障模式** | Heartbeat Checkpoint Silent Re-dispatch |
| **触发条件** | Tool call exceeds ~180s on LangGraph Cloud; BG_JOB_HEARTBEAT=120 hardcoded |
| **观察症状** | Duplicate runs with identical arguments appear; 2-3x redundant work and cost; no preceding model node in trace |
| **根因** | Unacknowledged heartbeats during long tool calls trigger stale run detection; sweep re-dispatches from last checkpoint while original still runs |
| **检测方法** | CADVP CC-8 execution boundary monitoring for duplicate tool invocations |
| **影响** | 2-3x cost multiplication; data corruption from concurrent writes; users cannot disable this behavior |
| **缓解措施** | Allow BG_JOB_HEARTBEAT customization; detect liveness during long awaits; skip re-dispatch if original still active |
| **证据** | [https://github.com/langchain-ai/langgraph/issues/7417](https://github.com/langchain-ai/langgraph/issues/7417) |
| **状态** | Open |
| **分级** | Tier B |
| **分类** | execution |
| **证据等级** | reported |
