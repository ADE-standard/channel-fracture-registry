# CF-0106: CrewAI after_kickoff_callbacks silently replaces result with None if callback doesn't return

| 字段 | 值 |
|------|-----|
| **CF-ID** | CF-0106 |
| **类别** | Callback result corruption |
| **影响系统** | CrewAI |
| **ADE层级** | L3 |
| **故障模式** | Callback Return Coercion |
| **触发条件** | User writes side-effect callback (logging/metrics) without returning result |
| **观察症状** | result becomes None silently; downstream code crashes with AttributeError on None |
| **根因** | crew.py:1032-1033 assigns callback return value directly to result without checking if callback returned None vs intended value |
| **检测方法** | CADVP CC-3 callback execution result validation |
| **影响** | Silent result corruption; production crashes from unexpected None values in downstream processing |
| **缓解措施** | Check if callback returns None; preserve original result when callback is side-effect only |
| **证据** | [https://github.com/crewAIInc/crewAI/issues/6439](https://github.com/crewAIInc/crewAI/issues/6439) |
| **状态** | Open |
| **分级** | Tier B |
| **分类** | execution |
| **证据等级** | reported |
