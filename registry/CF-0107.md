# CF-0107: Google ADK: --a2a flag silently mounts no A2A routes due to shadowed json import

| 字段 | 值 |
|------|-----|
| **CF-ID** | CF-0107 |
| **类别** | Silent configuration failure |
| **影响系统** | Google ADK |
| **ADE层级** | L3 |
| **故障模式** | Flag Silent No-Op |
| **触发条件** | Run adk api_server --a2a with agent.json present; server starts but has zero A2A routes |
| **观察症状** | Server starts successfully but --a2a silently does nothing; 404 on all A2A endpoints; only one ERROR log line |
| **根因** | Shadowed json import in get_fast_api_app(): function-local 'import json' makes Python treat json as local for entire function; json.load() raises UnboundLocalError swallowed by try/except |
| **检测方法** | CADVP CC-2 runtime A2A route availability check after server start |
| **影响** | Silent deployment failure; user believes A2A is active but no agents are reachable via A2A protocol |
| **缓解措施** | Remove duplicate 'import json' inside function; add route count assertion on startup |
| **证据** | [https://github.com/google/adk-python/issues/6332](https://github.com/google/adk-python/issues/6332) |
| **状态** | Open |
| **分级** | Tier B |
| **分类** | configuration |
| **证据等级** | reported |
