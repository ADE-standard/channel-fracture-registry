# CF-0087

## Title
Agent execution loop crashes with TypeError when custom tool returns nested dict → no graceful fallback

## Category
Tool surface illusion

## Affected System
CrewAI

## Layer
L3

## Failure Mode
Nested Return Type Crash

## Trigger
Custom BaseTool returns deeply nested dict (e.g., REST API response)

## Observed Symptom
Agent crashes with TypeError; no graceful string conversion; execution aborted

## Root Cause
Execution loop expects flat/simple return types; nested dict triggers parsing failure without fallback

## Detection
CADVP CC-7 tool availability probe with nested response simulation

## Impact
False belief of arbitrary tool return support; nested responses silently crash agent

## Mitigation
Add recursive string coercion for nested dicts in tool output parsing

arxiv:2606.04896
