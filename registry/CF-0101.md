# CF-0101: Checkpoint deserialization silently loses state when JsonPlusSerializer encounters unknown types

| 字段 | 值 |
|------|-----|
| **CF-ID** | CF-0101 |
| **类别** | State loss |
| **影响系统** | LangGraph |
| **ADE层级** | L2 |
| **故障模式** | Checkpoint Deserialization Silent Loss |
| **触发条件** | JsonPlusSerializer encounters unknown custom types during checkpoint deserialization |
| **观察症状** | Agent resumes with dict instead of typed object; downstream nodes produce incorrect results with no error |
| **根因** | fallback returns raw dict instead of raising DeserializationError when type annotation fails to deserialize |
| **检测方法** | CADVP CC-4 content hash verification on checkpoint round-trip |
| **影响** | Silent state corruption in long-running stateful workflows; cascading failures before detection |
| **缓解措施** | Raise DeserializationError instead of returning raw dict; add type registry for custom types |
| **证据** | [https://github.com/langchain-ai/langgraph/issues/7066](https://github.com/langchain-ai/langgraph/issues/7066) |
| **状态** | Fixed |
| **分级** | Tier B |
| **分类** | state |
| **证据等级** | reported |
