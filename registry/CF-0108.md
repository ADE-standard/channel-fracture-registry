# CF-0108: AutoGen MCP tool errors propagate and abort entire agent run instead of being isolated

| 字段 | 值 |
|------|-----|
| **CF-ID** | CF-0108 |
| **类别** | Error propagation |
| **影响系统** | AutoGen |
| **ADE层级** | L3 |
| **故障模式** | Tool Error Boundary Violation |
| **触发条件** | Single MCP tool fails during multi-agent scenario; error propagates up as fatal exception |
| **观察症状** | Entire agent run aborts from single tool failure; no ToolResult with is_error=True for agent reasoning |
| **根因** | MCP tool integration has no error isolation; tool exceptions bypass ToolResult wrapper and propagate as unhandled errors |
| **检测方法** | CADVP CC-7 tool availability probe with error injection |
| **影响** | Entire agent run destroyed by single tool failure; no recovery path; defeats declarative tool registration |
| **缓解措施** | Wrap MCP tool calls in error isolation; return ToolResult with is_error=True instead of raising |
| **证据** | [https://github.com/microsoft/autogen/issues/7851](https://github.com/microsoft/autogen/issues/7851) |
| **状态** | Open |
| **分级** | Tier B |
| **分类** | execution |
| **证据等级** | reported |
