# CF-0085

## Title
$defs not preserved in AIFunctionKernelFunction → MCP tool schema incomplete to LLM

## Category
Tool surface illusion

## Affected System
Semantic Kernel

## Layer
L2

## Failure Mode
Schema Reference Erasure

## Trigger
MCP tool with $ref/$defs in function schema definition

## Observed Symptom
LLM receives incomplete schema; references silently dropped; tool appears to have different interface

## Root Cause
AIFunctionKernelFunction serializes schema without preserving definitions/$defs section

## Detection
CADVP CC-7 tool availability probe with schema fidelity check

## Impact
False belief of complete tool interface; LLM operates on corrupted schema

## Mitigation
Preserve complete JSON Schema including $defs during function serialization

arxiv:2606.04896

https://github.com/microsoft/semantic-kernel/issues/13447

## Status
Fixed
