# CF-0072

## Title
ChatGroq mutates per-generation token_usage in place → silent clobber across generations

## Category
Memory blindness

## Affected System
LangChain

## Layer
L2

## Failure Mode
Token Usage In-Place Mutation

## Trigger
ChatGroq._combine_llm_outputs processing multiple generations

## Observed Symptom
Token usage counts silently overwritten; per-generation accounting corrupted

## Root Cause
In-place mutation of shared token_usage dict across generations; last write wins

## Detection
CADVP CC-1 memory availability check on generation metadata

## Impact
False belief of accurate token accounting; usage data silently corrupted

## Mitigation
Deep copy token_usage before mutation per generation

## Reference
arxiv:2606.04896

Confirmed
