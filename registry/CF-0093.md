# CF-0093

## Title
Individual MCP tool failure aborts entire agent run → no error isolation between tools

## Category
Tool surface illusion

## Affected System
AutoGen

## Layer
L3

## Failure Mode
Tool Failure Cascade

## Trigger
Single MCP tool fails during multi-tool agent run

## Observed Symptom
Entire agent run aborted; other independent tools never execute; one failure cascades

## Root Cause
No per-tool error isolation; single MCP tool exception propagates to agent-level abort

## Detection
CADVP CC-7 tool availability probe with individual failure simulation

## Impact
False belief of tool isolation; one broken tool gatekeeps entire agent workflow

## Mitigation
Isolate MCP tool errors: catch per-tool, report failure, continue with remaining tools
