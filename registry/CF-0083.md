# CF-0083

## Title
AgentGroupChat fails with 'Unknown response format dict' when Azure Monitor enabled → monitoring silences agents

## Category
Env contamination

## Affected System
Semantic Kernel

## Layer
L4

## Failure Mode
Monitoring-Induced Agent Failure

## Trigger
Azure Monitor (configure_azure_monitor) enabled with AzureAIAgent in AgentGroupChat

## Observed Symptom
All agent executions fail with ValueError; multi-agent flow completely blocked

## Root Cause
Azure Monitor injects dict-format response into agent pipeline; AgentGroupChat expects typed response

## Detection
CADVP CC-1 guard execution counter with external instrumentation active

## Impact
False belief of monitoring transparency; enabling observability silently breaks agent execution

## Mitigation
Isolate monitoring pipeline from agent response format expectations

arxiv:2606.04896

Fixed
