# CF-0062

## Title
ToolCallResult auto-coerces structured outputs to str() → losing type info

## Category
Tool surface illusion

## Affected System
AutoGen

## Layer
L3

## Failure Mode
Type Information Loss via Auto-Coercion

## Trigger
Tool returns dict/Pydantic/dataclass to AutoGen agent

## Observed Symptom
Downstream agents receive string instead of structured data

## Root Cause
ToolCallResult silently coerces structured return to str() without preserving type

## Detection
CADVP CC-7 tool availability probe with type fidelity

## Impact
False belief of structured inter-agent communication; string parsing instead of typed access

## Mitigation
Preserve original return type in ToolCallResult payload

arxiv:2606.04896
