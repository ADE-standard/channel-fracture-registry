# CF-0105: LangGraph checkpoint serialization produces 85% storage bloat and silent pass failures with no opt-out

| 字段 | 值 |
|------|-----|
| **CF-ID** | CF-0105 |
| **类别** | Silent correctness failure |
| **影响系统** | LangGraph |
| **ADE层级** | L2 |
| **故障模式** | Serializer Silent Pass Corruption |
| **触发条件** | Non-primitive types in checkpoint state; serialization passes silently but produces corrupted/incomplete data |
| **观察症状** | 85% storage bloat + 37.8% token overhead; 'silent pass' failures where serializer claims success but data is wrong |
| **根因** | JsonPlusSerializer has no structural validation; corrupt agent payloads pass through to LLM unchecked |
| **检测方法** | CADVP CC-4 content hash validation on serializer round-trip |
| **影响** | Wasted storage/tokens; silent data corruption at the serializer boundary; no opt-out or alternative serializer |
| **缓解措施** | Add structural validation hook at serde boundary; expose pluggable serialization protocol |
| **证据** | [https://github.com/langchain-ai/langgraph/issues/7714](https://github.com/langchain-ai/langgraph/issues/7714) |
| **状态** | Open |
| **分级** | Tier C |
| **分类** | state |
| **证据等级** | reported |
