# CF-0100

## Title
get_num_tokens_from_messages raises NotImplementedError for OpenAI o1 and o3 models → token counting silently broken

## Category
Tool surface illusion

## Affected System
LangChain

## Layer
L2

## Failure Mode
Token Counter Model Gap

## Trigger
Token counting for o1/o3 model messages

## Observed Symptom
NotImplementedError raised; all token-aware logic (truncation, budget) silently fails

## Root Cause
Token counting implementation doesn't cover o1/o3 model families; no fallback to generic counter

## Detection
CADVP CC-7 tool availability probe with o1/o3 model token counting

## Impact
False belief of token budget enforcement; truncation and cost tracking silently broken for o-series

## Mitigation
Add o1/o3 token counting support or fallback to tiktoken generic encoding

arxiv:2606.04896
