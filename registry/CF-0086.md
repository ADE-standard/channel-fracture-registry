# CF-0086

## Title
Async task LLM failure silently freezes flow → no exception, no log, process hangs

## Category
Hook not firing

## Affected System
CrewAI

## Layer
L3

## Failure Mode
Async LLM Failure Silent Freeze

## Trigger
LLM call fails during async task execution in production crew

## Observed Symptom
Process hangs indefinitely with no exception or log; full day debugging to locate

## Root Cause
Async task LLM failure path swallows exception; no callback/retry/halt mechanism triggers

## Detection
CADVP CC-3 hook execution counter on async failure path

## Impact
False belief of fault-tolerant execution; production stuck without operator awareness

## Mitigation
Add exception re-raise + logging to async LLM failure path; explicit timeout
