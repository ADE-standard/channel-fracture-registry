# CF-0056

## Title
GraphInterrupt not re-raised in awrap_tool_call → signal silently swallowed

## Category
Hook not firing

## Affected System
LangGraph

## Layer
L3

## Failure Mode
Interrupt Signal Suppression

## Trigger
GraphInterrupt raised during async tool execution

## Observed Symptom
Graph continues past interrupt boundary; operator believes it paused

## Root Cause
awrap_tool_call catches but does not re-raise GraphInterrupt

## Detection
CADVP CC-3 hook execution counter

## Impact
False belief of interrupt safety; runaway execution after pause point

## Mitigation
Re-raise GraphInterrupt in all wrapper exit paths

arxiv:2606.04896
