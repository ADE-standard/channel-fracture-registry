# CF-0073

## Title
ChatAnthropic drops thinking field from empty thinking blocks → replay fails with 400

## Category
Silent message loss

## Affected System
LangChain

## Layer
L3

## Failure Mode
Thinking Field Silent Truncation

## Trigger
ChatAnthropic streaming with signature_delta-only thinking blocks

## Observed Symptom
Empty thinking blocks lose thinking field; replay fails with 400 without indicating truncation

## Root Cause
Streaming handler drops thinking field from empty blocks instead of preserving for replay

## Detection
CADVP CC-4 end-to-end content hash

## Impact
False belief of replayable conversation; silent truncation causes downstream failure

## Mitigation
Preserve thinking field even on empty blocks for replay integrity

arxiv:2606.04896
