# CF-0091

## Title
cache_breakpoint injected into messages for non-Anthropic providers (Groq, OpenAI-compatible) → silent API errors

## Category
Env contamination

## Affected System
CrewAI

## Layer
L2

## Failure Mode
Provider-Specific Config Leakage

## Trigger
Non-Anthropic provider (Groq, OpenAI-compatible) used with cache_breakpoint

## Observed Symptom
cache_breakpoint field sent to providers that don't support it; API errors or ignored silently

## Root Cause
Anthropic-specific cache_breakpoint injected unconditionally; no provider capability check

## Detection
CADVP CC-2 config vs runtime provider capability diff

## Impact
False belief of cross-provider compatibility; Anthropic config silently leaks to other providers

## Mitigation
Strip cache_breakpoint for non-Anthropic providers based on capability check

arxiv:2606.04896
