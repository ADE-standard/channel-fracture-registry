# CF-0053

## Title
local_read() returns live mutable channel state → silent state corruption

## Category
Silent message loss

## Affected System
LangGraph

## Layer
L3

## Failure Mode
State Corruption via Mutable Reference

## Trigger
Conditional-edge router reads channel state via local_read() during graph execution

## Observed Symptom
Graph state silently mutates at routing points; downstream sees corrupted state

## Root Cause
local_read() returns direct reference to live mutable state instead of isolated copy

## Detection
CADVP CC-4 content hash across checkpoint boundaries

## Impact
False belief of deterministic execution; routing based on corrupted state

## Mitigation
Return deep copy from local_read() or enforce immutable channel views

arxiv:2606.04896
