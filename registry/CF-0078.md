# CF-0078

## Title
before/after_kickoff_callbacks do not support async callables in akickoff → silently skipped

## Category
Hook not firing

## Affected System
CrewAI

## Layer
L3

## Failure Mode
Async Callback Silent Skip

## Trigger
Async callable registered as kickoff callback in akickoff flow

## Observed Symptom
Async callback never fires; crew proceeds without pre/post validation; no error

## Root Cause
akickoff does not support async callables in callback chain; sync-only path skips async silently

## Detection
CADVP CC-3 hook execution counter on async path

## Impact
False belief of callback enforcement; pre/post validation silently skipped

## Mitigation
Add async callable support to akickoff callback chain
