# CF-0094

## Title
Bedrock connector rejects parallel tool calls → toolResult blocks not merged into single Converse message

## Category
Tool surface illusion

## Affected System
Semantic Kernel

## Layer
L3

## Failure Mode
Parallel Tool Call Rejection

## Trigger
Model calls multiple tools in one turn via Bedrock connector (Claude Sonnet)

## Observed Symptom
Follow-up Converse request rejected; toolResult ID mismatch error

## Root Cause
Function-calling loop does not merge multiple toolResult blocks into a single Converse message

## Detection
CADVP CC-7 tool availability probe with parallel tool call simulation

## Impact
False belief of parallel tool support; multi-tool calls silently fail on Bedrock

## Mitigation
Merge toolResult blocks into single Converse message before follow-up

arxiv:2606.04896
