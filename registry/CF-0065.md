# CF-0065

## Title
Fabricated-Observation recovery dead code → tool calls silently discarded

## Category
Hook not firing

## Affected System
CrewAI

## Layer
L3

## Failure Mode
Recovery Path Dead Code

## Trigger
LLM without stop-word support generates past tool call boundary

## Observed Symptom
Agent receives empty observation; no error; continues with fabricated assumptions

## Root Cause
process_llm_response() recovery block for no-stop-word models is dead code since #2483

## Detection
CADVP CC-3 hook execution counter on no-stop-word path

## Impact
False belief of tool execution; agent acts on fabricated observations

## Mitigation
Restore recovery logic or warn when models lack stop-word support
