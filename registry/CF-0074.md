# CF-0074

## Title
cache_control on tool_use content block silently dropped when matching tool_calls item exists

## Category
Config override

## Affected System
LangChain

## Layer
L3

## Failure Mode
Cache Control Silent Drop

## Trigger
cache_control set on tool_use content block with matching tool_calls

## Observed Symptom
Cache control directive ignored; repeated tool calls avoid cache, increasing latency/cost

## Root Cause
Matching tool_calls item causes cache_control to be silently dropped from content block

## Detection
CADVP CC-2 config vs runtime cache control verification

## Impact
False belief of caching; repeated tool calls incur full cost without awareness

## Mitigation
Merge cache_control from content block with tool_calls matching
