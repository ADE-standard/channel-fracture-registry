# CF-0075

## Title
SummarizationMiddleware preserves orphan ToolMessages → provider 400 on next call

## Category
Silent message loss

## Affected System
LangChain

## Layer
L3

## Failure Mode
Orphan Message Preservation

## Trigger
SummarizationMiddleware._find_safe_cutoff_point on conversation with tool messages

## Observed Symptom
Orphan ToolMessages (without matching HumanMessage) survive cutoff; next LLM call fails 400

## Root Cause
_find_safe_cutoff_point preserves ToolMessages but not their paired HumanMessages

## Detection
CADVP CC-4 content hash across summarization boundary

## Impact
False belief of safe summarization; orphan messages silently corrupt next call

## Mitigation
Remove orphan ToolMessages when paired HumanMessage is cut off
