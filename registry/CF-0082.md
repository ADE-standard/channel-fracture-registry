# CF-0082

## Title
ReasoningEffort setting lost when invoking orchestration → handoff drops model config

## Category
Config override

## Affected System
Semantic Kernel

## Layer
L3

## Failure Mode
Model Config Silent Drop on Handoff

## Trigger
Agent with ReasoningEffort='high' handed off in orchestration

## Observed Symptom
HTTP request missing reasoning_effort field; model runs with default reasoning level

## Root Cause
Handoff orchestration does not propagate per-agent ReasoningEffort to downstream LLM calls

## Detection
CADVP CC-2 config vs runtime diff on model parameters

## Impact
False belief of high-reasoning execution; agent silently downgraded to default

## Mitigation
Propagate all per-agent model settings through orchestration handoff
