## 背景 (Background)

Cron memory contract ambiguous: read always, write behind opt-in flag creates silent expectation gap

**故障表现**: Maintainer aggregation: read by default, write with opt-in

---

## 故障链 (Failure Chain)

1. **系统状态**: 组件处于正常运行状态，操作者预期一切正常
2. **触发条件**: 配置或环境条件触发边界处的数据/状态丢失
3. **静默失效**: 数据在 `Cron↔Memory` 边界处被丢失/阻塞，调用方未收到错误
4. **下游影响**: 后续操作基于不完整/错误的状态继续执行
5. **可观测性缺口**: 标准监控（日志、退出码、仪表盘）未能反映实际故障

---
