## About

The **ADE Channel Fracture Registry** is a public failure knowledge base for
Multi-Agent Reliability Engineering. It catalogs real-world instances of
*Channel Fracture* — a class of silent, cross-boundary failures where data
or state is lost at system boundary crossings while the system reports success.

Channel Fracture was first characterized in:
> Liu, D. *Channel Fracture in Long-Horizon LLM-Driven Multi-Agent Systems.*
> arXiv:2606.04896, 2026.
